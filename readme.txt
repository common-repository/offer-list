Now you can build a list of blocks representing your offers. Users can use search input to find these positions easly. 

== Description ==
Create a list of blocks and show it to your users via shortcode.
Users can search through blocks using search input.
You can make only a few blocks visible and hide the rest of them until user clicks "show all".

== Installation ==
Simply extract the zip file and drop the contents into the wp-content/plugins/ directory of your WordPress installation. Then activate the plugin from the plugins page.
